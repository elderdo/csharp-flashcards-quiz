What does AsNoTracking do in EF Core?	Disables change tracking for read-only queries, reducing overhead.
When should you prefer async EF methods like ToListAsync?	For I/O-bound database calls to avoid blocking threads.
What is eager loading in EF Core?	Loading related entities up front via Include/ThenInclude.
What can happen with lazy loading in high-traffic scenarios?	N+1 query problems and unexpected extra round-trips.
How can you inspect SQL generated by EF Core?	Use ToQueryString() or enable EF Core SQL logging.
What is a compiled query in EF Core?	A cached query delegate to reduce repeated query compilation overhead.
Why project with Select into DTOs?	Fetch only required columns and reduce materialization cost.
What does SaveChanges do regarding transactions?	It wraps operations in a transaction when needed for consistency.
How do migrations help EF Core projects?	They version and apply schema changes consistently.
What is a common optimization for bulk inserts/updates?	Batch operations and minimizing tracked entity overhead (or specialized bulk libraries).
